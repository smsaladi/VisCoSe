VisCoSe - Visualization and Comparison of consensus Sequences
Michael Spitzer, Georg Fuellen, Paul Cullen, and Stefan Lorkowski
University of Muenster, Germany

ABSTRACT
VisCoSe is a WWW service and a stand-alone command line Perl script for visualizing and comparing consensus sequences of protein and nucleotide sequences. The input is a set of aligned or unaligned sequences, which in the latter case is automatically aligned. By defining groups (WWW service and stand-alone version) or by submitting the input in the form of multiple datasets (stand-alone version only), the user may define subsets of sequences. VisCoSe generates the consensus for each subset and an alignment of the consensus sequences that is visualized using HTML. Optionally, VisCoSe can additionally calculate consensus sequences and alignment visualizations on the basis of "reduced alignments". The user may choose one of a set of reduced amino acid alphabets and VisCoSe will then translate all datasets/subsets and calculate the consensus sequence and conservation rates for these data. The visualization of each single consensus sequence along with its conservation rates for each position and the underlying alignment can be accessed via hyperlinks. VisCoSe is the only interface currently available that simultaneously calculates consensus sequences of multiple datasets and automatically compares these consensus sequences.

DESCRIPTION
VisCoSe provides an easy way for calculating and visualizing consensus sequences of nucleotide sequences and of large protein sequences and is the only tool currently available that calculates and visualizes the comparison of multiple consensus sequences. VisCoSe allows both the visualization of consensus sequences and conservation rates for single as well as for multiple alignments/datasets. Unaligned datasets are aligned by VisCoSe using the multiple alignment program fftnsi from the MAFFT library, version 3.85 (Katoh et al., 2002).

VisCoSe is available as a stand-alone command line Perl script and a HTML-based web application. It is not necessary to handle large and complex parameter lists. The user can choose from a reasonable set of visualizing parameters (see below). The resulting graphical presentation of the consensus sequences and the conservation rates of each amino acid are clear and straightforward (see examples). Another advantage is the presentation of the results by HTML. Since both the sequences and the alignments are shown continuously in full length by default, it is simple to identify highly conserved regions and assign them to the sequence. Moreover, the user has the possibility to choose one of a set of reduced amino acid alphabets (see here for a description). In that case, VisCoSe will translate the alignment(s) into reduced code and additionally calculate consensus sequences and (if specified) alignment visualisations. We recommend the WWW browser Mozilla because of its fast scrolling abilities and high compatibility.

The use of VisCoSe is simple and the following input options are available:

(1)
It is possible to use a single unaligned dataset of protein or nucleotide sequences in FASTA format. In this case, VisCoSe uses MAFFT to align the dataset. Multiple datasets can be "simulated" by using a single dataset as input and specifying "groups", which divide the dataset into two or more single subsets. VisCoSe then produces sequence alignments for each subset. In this case, the resulting consensus sequences are subjected to multiple alignment.
(2)
A pre-calculated alignment of a protein or nucleotide sequence set can be entered that has been produced using a method other than MAFFT, such as CLUSTAL W or DIALIGN. Here also, groups can be defined to specify independent subsets.
(3)
Using the stand-alone version, it is possible to enter two or more different datasets consisting of either aligned or unaligned sequences. In the latter case, the sequences are automatically aligned.

VisCoSe automatically calculates the consensus for each alignment following the relative majority rule and compiles the consensus into FASTA format. The grade of conservation at each position is compiled into a color scheme ranging from dark blue (indicating no conservation) to red (indicating high conservation) and a histogram indicating grades of conservation.

To access the results generated by VisCoSe, the following output is produced:

(1)
A zipped TAR archive containing all results and intermediate data such as aligned data subsets.
(2)
The sequence dataset and user-defined groups of sequences as provided by the user.
(3)
The consensus of each input dataset (or user-defined group) in FASTA format and in colorized HTML format. If a reduced alphabet was chosen, VisCoSe additionally calculates the consensus sequence for the resulting reduced alignment.
(4) The alignment of the consensus sequences colorized and illustrated according to their column-wise conservation rate. If a reduced alphabet was chosen, VisCoSe additionally visualizes the resulting reduced alignment.
(5) The alignment and consensus sequences in a reduced code giving information on chemical properties of the amino acid residues if chosen.

OPTIONS
The user controls the VisCoSe output using a few options, mainly affecting realigning behavior of VisCoSe, the presentation of the alignment on which a consensus is based and the conservation graph appearance. The options available are described in detail on the help page.

PLANNED ENHANCEMENTS
We plan to implement the following enhancements of the current version of VisCoSe:
· Different methods for multiple alignment such as CLUSTAL W (Thompson et al., 1994), DIALIGN (Morgenstern, 1999) and TCOFFEE (Notredame et al., 2000).
· Different methods for calculating consensus sequences such as described by [Keith et al (2002)] and [Lee (2003)].
