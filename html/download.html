REQUIREMENTS
 VisCoSe was designed to run under Linux / Unix. We cannot give any support concerning installation/usage of VisCoSe under the Microsoft Windows OS.
The standalone script of VisCoSe needs the following things to run:
- Perl 5.6.1 or higher (tested with 5.8, not tested with versions <5.6.1, so functionality can't be guaranteed in the latter case)
- Bit::Vector library by Steffen Beyer
- MAFFT installed in a way that the executables and scripts (fftns, fftnsi, ...) are present and accessible in $PATH
- VisCose itself

INSTALLATION
There are detailed installation instructions inside the VisCoSe-archive, but in general the installation consists of three parts:
- MAFFT installation
- Installation of VisCoSe and its Perl modules
- Adjustment of the search path for these modules in the main VisCoSe Perl script
