VisCoSe - Visualization and Comparison of consensus Sequences
Example input and output of a sample VisCoSe run

An example input alignment can be found here. By saving it to your local harddisk you can provide it as an input alignment to our VisCoSe webservice.
It's an example unaligned dataset compiled from some human ABC proteins of the subfamilies A, B and C and consists of a total of 35 protein sequences (11 ABC proteins of the subfamily A, 10 proteins of the subfamily B, 14 proteins of the subfamily C).
Group ranges for VisCoSe were "1-11;12-21;22-35", were "1-11" denotes the proteins of the subfamily A, "12-21" those of the subfamily B and "22-35" those of the subfamily C.
Therefore output of VisCoSe in this example case consists of the following files:

abc_a_b_c.fasta the original dataset as provided by the user (here: unaligned!)
consensi_aligned.fasta aligned consensus sequences of the 3 subgroups "1-11;12-21;22-35", FASTA format
consensi_aligned.html aligned consensus sequences, HTML format
consensi_unaligned.fasta unaligned consensus sequences, FASTA format
consensus_grp_1-11_abc_a_b_c.fasta consensus sequence of group "1-11", FASTA format
consensus_grp_1-11_abc_a_b_c.html consensus sequence of group "1-11", HTML format
consensus_grp_12-21_abc_a_b_c.fasta consensus sequence of group "12-21", FASTA format
consensus_grp_12-21_abc_a_b_c.html consensus sequence of group "12-21", HTML format
consensus_grp_22-35_abc_a_b_c.fasta consensus sequence of group "22-35", FASTA format
consensus_grp_22-35_abc_a_b_c.html consensus sequence of group "22-35", HTML format
grp_1-11_abc_a_b_c.fasta partial dataset for group "1-11"
grp_1-11_abc_a_b_c.fasta_aligned partial dataset for group "1-11", aligned
grp_12-21_abc_a_b_c.fasta partial dataset for group "12-21"
grp_12-21_abc_a_b_c.fasta_aligned partial dataset for group "12-21", aligned
grp_22-35_abc_a_b_c.fasta partial dataset for group "22-35"
grp_22-35_abc_a_b_c.fasta_aligned partial dataset for group "22-35", aligned

All files containing color formatted HTML content like consensus comparisons or visualisation of conservation rates are presented with bold-faced filenames.
