Amino acid alphabets

VisCoSe allows the additional use of a specific reduced amino acid alphabet. If the user chooses an alphabet other than "None / Default 20 aa" VisCoSe translates the input alignment (after group-splitting and realignment, if necessary) into the simplified alphabet and calculates a consensus sequence along with conservation rates of the resulting alignment.
The alphabets from which the user can choose are the following:
